# Notification Implementation

This project uses the OpenClaw CLI to send DingTalk messages for notifications instead of WeCom webhooks. 

## Environment Variables
- **NOTIFY_USER_ID**: Your DingTalk user ID for notifications.

Make sure to have the OpenClaw dependency installed. 

## Note
The OpenClaw CLI is used for sending notifications directly via DingTalk instead of the previously mentioned WeCom webhooks.
